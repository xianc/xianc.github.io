About

Gunmetal was created on April 18th 2015 as an responsive design excercise that incorporates a carousel, and fade-in messaging (as user scrolls).

The theme of Gunmetal is inspired by the orange tones of rust and the greys of metal. It is a single page responsive site with top navigation bars that jumplink down the page to its corresponding sections. The HTML, CSS & Javascript is written from scratch.
